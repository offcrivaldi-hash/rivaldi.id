# Website Cerita untuk AdSense

Website ini adalah situs web berbasis cerita yang dirancang untuk dioptimalkan dengan Google AdSense. Konten berupa cerita panjang dan menarik untuk meningkatkan waktu baca dan impresi iklan.

## Fitur

- Halaman beranda dengan daftar cerita
- Halaman cerita individu dengan konten panjang
- Placeholder untuk AdSense yang siap diganti dengan kode asli
- Desain responsif dan sederhana

## Cara Menjalankan

1. Buka file `index.html` di browser web Anda.
2. Atau, gunakan server lokal seperti:
   - Python: `python -m http.server 8000`
   - Node.js: Install `http-server` dan jalankan `http-server`

## Penggunaan

- **Ganti Placeholder AdSense**: Cari komentar "Placeholder for AdSense" dan ganti dengan kode AdSense Anda yang sebenarnya. Pastikan untuk mengganti `ca-pub-XXXXXXXXXXXXXXXX` dengan publisher ID Anda dan `XXXXXXXXXX` dengan ad slot ID.
- **Ganti Konten Cerita**: Konten cerita saat ini adalah placeholder. Ganti dengan cerita asli Anda yang panjang (minimal 1000 kata per cerita) untuk optimasi AdSense.
- **Optimasi SEO**: Tambahkan meta tag, judul, dan deskripsi yang relevan untuk meningkatkan peringkat pencarian.
- **Monetisasi**: Pastikan situs memenuhi kebijakan AdSense, termasuk konten original dan berkualitas tinggi.

## Lisensi

Konten placeholder ini untuk demonstrasi. Ganti dengan konten asli Anda. Kode sumber bebas digunakan untuk tujuan non-komersial.

## Catatan

Website ini dibuat sebagai template dasar. Untuk produksi, pertimbangkan menggunakan framework seperti Next.js untuk performa yang lebih baik.
